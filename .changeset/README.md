# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the changesets repository](https://github.com/changesets/changesets).

## Adding a changeset

When you make changes that should be included in a release, run:

```bash
npx changeset
```

This will prompt you to select the type of version bump (patch, minor, major) and add a summary of the changes.

The changeset file will be committed with your PR and consumed when the "Version Packages" PR is merged.
